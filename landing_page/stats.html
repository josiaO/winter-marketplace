Login click insights

Tracks clicks on Log in and other CTAs that send visitors to the app sign-in page. Each row is a short fingerprint (not a name): browser or OS summary plus a stable visitor token derived on the server. Set LANDING_LOGIN_STATS_SECRET in apps/frontend/.env.local and restart the Next.js dev server. Open this file over http:// (e.g. npx serve landing_page) if your browser still blocks requests from a file:// tab.
